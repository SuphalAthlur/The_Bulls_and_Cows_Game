TRIVIA

  • A computer program 'moo', written by Frank King in 1968 on the Cambridge University Titan mainframe, was among the first Bulls and Cows computer implementations.

  • This inspired a similar program written in 1970 by J.M. Grochow at MIT in the PL/I computer language for the Multics OS.

  • The program maintained a league table of players' scores, and protecting the integrity of this league table became a popular case study for researchers into computer security.

  • It is proved that any number can be solved within seven turns. The average minimal game length is 5.21 turns.
